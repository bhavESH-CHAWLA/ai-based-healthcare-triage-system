# Frontend (React)

## Setup
1. Go to `frontend/`
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env`
4. Start dev server: `npm run dev`

## Environment
- `VITE_API_BASE_URL` should point to backend server (default: `http://localhost:5000`)

## Backend requirement
Run backend from `server/` before using frontend.
